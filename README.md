# 🤖 AI-Powered Leave Permission Email Generator

## Overview
An n8n automation workflow that generates and sends 
professional leave/exam permission emails using AI.

## Workflow
On Form Submission → Gemini AI → Gmail → JavaScript

## Tools Used
- n8n (Automation)
- Google Gemini (AI Email Generation)
- Gmail (Email Sending)

## How to Use
1. Import workflow.json into n8n
2. Connect Gmail credentials
3. Connect Gemini AI credentials
4. Activate the workflow
5. Submit the form
```

## Step 5: Import Workflow (for others)

Anyone can use your workflow by:
1. Downloading your `workflow.json`
2. Opening n8n → **"Import from file"**
3. Adding their own credentials
